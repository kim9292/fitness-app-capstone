# Frontend Planning

## Main Pages
- Home / Dashboard
- Register / Login
- Workouts (library, planner, details)
- Habits tracker
- Meals / Nutrition
- Progress / Stats
- AI Chat (floating widget)

## Components
- ProminentHeader (gradient, title, stats)
- WorkoutCard, HabitCard, MealCard
- AIChatBox, FloatingChatWidget
- Forms: login, register, workout/meal entry

## User Flows
- Register → Onboarding → Dashboard
- Add workout/meal/habit → View progress
- Ask AI chat for help or advice

## UI/UX Goals
- Clean, modern, mobile-friendly
- Consistent color theme and spacing
- Clear CTAs and navigation
